# DDP Branding Assets

Dit is de officiële huisstijl van de Democratische Digitale Partij.

## Mappen
- **icons/** – logo’s (icon, stacked, horizontal)
- **favicons/** – browser en app icons
- **social/** – Open Graph social banners
- **palette/** – kleurdefinities in CSS en JSON

## Gebruik
Plaats `logo.svg` in de navigatie van de website.

Gebruik `logo-horizontal.svg` voor social previews en headers.

Favicon-bestanden gaan automatisch werken wanneer geplaatst in:
`<link rel="icon" ...>`
